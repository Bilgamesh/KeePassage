This is a custom binary of [pcsc-mini](https://github.com/kofi-q/pcsc-mini) built with a [fork of tokota](https://github.com/Bilgamesh/tokota) to support `keepassage.exe` executable on Windows. It is required due to a limitation of pcsc-mini.

For more details, see:
- https://github.com/kofi-q/pcsc-mini/issues/13
- https://github.com/kofi-q/pcsc-mini/issues/5
